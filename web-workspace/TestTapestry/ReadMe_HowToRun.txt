How to run the sample:

0. Make sure you'v already installed maven 2.0.5

1. Double check the path in tapestry.xml and deploy_home.bat or deploy_office.bat

2. Put tapestry.xml to $TOMCAT/conf/Catalina/localhost

3. Double click on deploy_home.bat or deploy_office.bat to build your app if any source code changes happen.

4. Restart Tomcat and visit "http://localhost:8080/t5first" to see the changes.
